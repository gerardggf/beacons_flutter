## 1.0.0

* Initial release
* Support for Android (API 21+) and iOS (13.0+)
* Beacon protocol detection: iBeacon, Eddystone (UID, URL, TLM, EID), AltBeacon
* Real-time BLE scanning with detailed beacon information
* RSSI, TX Power, Manufacturer Data, Service UUIDs, and Service Data
* Stream-based API for continuous beacon monitoring
* Permission handling for Bluetooth and Location
* Hexadecimal and ASCII data display support
